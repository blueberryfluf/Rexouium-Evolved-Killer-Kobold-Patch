Folder
------
Assets/Blueberrys_Flufs_Patcher

What this is
------------
An Editor tool that patches Killer Kobold carrier animations for Rexouium Evolved
(maw open, gulp, delayed belly / stomach size, burps, lip sync, etc.).

Requirements
------------
1. Unity project with VRChat SDK
2. Rexouium Evolved avatar
3. Killer Kobold vore system already in the project (Assets/VoreSystem)

Install
-------
1. Assets → Import Package → Custom Package…
2. Pick BlueBerry-RexouiumCarrierPatch.unitypackage
3. You should see Assets/Blueberrys_Flufs_Patcher
4. Open Tools → BlueBerry → Rexouium Evolved Carrier Patch
5. Select your avatar base, then click Patch Avatar

Notes
-----
This package only ships the patcher tool + icons. It does NOT include Killer Kobold
or Rexouium Evolved content. Re-run Patch after updating Killer Kobold assets.
